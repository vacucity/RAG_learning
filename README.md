## Introduction 

I first learned about RAG from YouTube [videos](https://www.youtube.com/watch?v=wd7TZ4w1mSw&t=35s).  

Due to recent upgrades to the langchain package, the original code provided in the video description no longer worked properly, so I made corrections to the code.

Additionally, since using OpenAI’s API can be quite expensive, I switched to Tongyi as a more cost-effective alternative to GPT.  

The [Documents](https://github.com/vacucity/RAG_learning/tree/main/Document) summarize the courses from [费曼学徒冬瓜](https://www.bilibili.com/video/BV1hj5DzyE5d?vd_source=44c1b036a14b68e5e29a1ccaad1c731b)

Some cases may be updated in the future.
